Running the Project Looking Glass Developer's Release on Microsoft Windows

Project Looking Glass runs on Windows!

Running Project Looking Glass (LG3D) on Windows is very easy :-) The core part of LG3D is built using platform independent Java technology which means that it can run on many platforms. So, it runs on Windows (although there are some limitations). In order to simplify the installation process significantly, the LG3D Installer exe has been released. This document describe how to install and run LG3D on your Windows system using the installer.

1. Platform Requirements

First check to make sure your system meets the following minimal requirements:

Component Requirements
Operating System Tested on Windows XP and Windows 2000
CPU 1.4 GHz or faster recommended
RAM 512MB recommended
Graphics Card 3D accelerated graphics card, with at least 16MB VRAM and driver support for OpenGL, version 1.3 or greater. 16bit or 32bit color.

Tested cards:
  • ATI MOBILITY RADEON
  • ATI Radeon 9800 Pro 128MB
  • NVIDIA GeForce4/5200/6600/6800
  • Intel 855GM/915G

If you are successful in getting LG3D to run on other cards, or want to report problems with specific cards, please let us know using the discussion forum

Disk Space 250MB

Limitations on Windows

  • No Windows Native App in 3D Space...

    Although LG3D for Linux and Solaris includes support for running native applications within the LG3D desktop, this feature is not available for the Windows release. This menas that it is not possible to run native Windows applications within LG3D. You can, however, develop and run and develop LG3D applications, such as the CD viewer and other LG3D Java applications. In other words, the Windows version is designed for experience LG3D and app development, but not for replacing Windows desktop.

2. Downloading LG3D Installer

LG3D Installer for Windows installs everything you need to run LG3D, including JRE and Java 3D. Although it is pretty big, it simplifies your installation process (so please bear with the download time!).

The Installer can be downloaded from the following link:

  • LG3D Release 1.0.0 Beta 1 Windows Installer (19 October 2006)

Download the Installer exe file. Other builds can be found in the Stable build section on the binary build page .

3. Installing LG3D

The LG3D Installer installs LG3D and all the necessary components including JRE and Java 3D. It also adds an LG3D folder to the Start menu. Note that the installer puts JRE and Java 3D in an LG3D specific folder. So, the installation doesn't alter existing Java environment you might already have.

  1. Locate the saved Installer exe file and execute it.

  2. Windows may display a dialog window that mentions "Security Warning" regarding runing this exe file because of "Unknown Publisher". Click "Run" in the dialog window in that case.

  3. The installer shows License Agreement. Read through it, and click "I Agree" button if you agree with it.

  4. The insaller askes you an Installation Folder. Select a foler to install LG3D, then press the "Install" button.

  5. It shows installation progress. When completed, click the "Close" button.

After successful installation, you should see the "LG3D Project" entry in the Start menu, that includes LG3D launcher, uninstaller, and a shortcut to the LG3D website.

4. Running LG3D

After successful installation, you can launch LG3D from the Start menu:

Start -> Programs -> LG3D Project -> Start LG3D Project

Click the launcher. After a few seconds, the LG3D window should appear. Congratulations! If you have problems, please refer to the Troubleshooting section for assistance.

A Few FAQs

  • How to start LG3D with the console window minimized?
    1. Right click on the LG3D launcher. It brings up a menu list.
    2. Select "Properties" from the menu. It brings up a Properties configuration window.
    3. From the "Shortcut" tab in the Properties window, locate the "Run:" filed. Select "Minimized" from the pull-down selection list.
    4. Click "OK" in the Properites window

  • How to start LG3D in Fullscreen mode?
    1. Right click on the LG3D launcher. It brings up a menu list.
    2. Select "Properties" from the menu. It brings up a Properties configuration window.
    3. From the "Shortcut" tab in the Properties window, locate the "Target:" filed. Add -f at the end (after the last double-quotation). It should look like this:
      ...\bin\lg3d-run.bat" -f
    4. Click "OK" in the Properites window

  • How to further configure the LG3D window?
    • Check out the Resizing and Configuring LG3D Window document .

5. Troubleshooting

Screen update is very slow

The graphics card on your machine may not be fast enough for running LG3D. Try "Medium (16 bit)" color mode by: Right-click on the background ->select "Properties" ->select "Settings" in the "Display Properties" window ->select "Medium (16bit) in the "Color quality" selection. You may also consider to update your graphics card driver. Please google for the details.

See also: Project Looking Glass Developer's Release Notes

Additional information on current problems and issues can be found under lg3d-core Project Tools -> Issue tracker. You can search for issues and defects or find a specific issue.
