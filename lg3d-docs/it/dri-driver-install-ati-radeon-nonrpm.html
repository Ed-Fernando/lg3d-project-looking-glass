Installazione driver DRI per ATI Radeon


Queste istruzioni spiegano come installare il driver Direct Rendering Infrastructure (DRI) per schede grafiche ATI Radeon per abilitare l' estensione per il rendering diretto richiesta da Project Looking Glass.

Istruzioni per l'installazione

Questa è una guida per installare i driver binari ATI su distribuzioni non basate su rpm come Slackware. Si farà riferimento ai comandi necessari su Slackware, ma molti passi possono essere comunque applicati alle altre distribuzioni.

  1. Da root:

    % su -
    Password: inserire la password di root
    #

  2. Installare i sorgenti del kernel dal pacchetto kernel-source della distribuzione:

    # installpkg kernel-source-2.4.26-noarch-4.tgz

    Nota: Nella distribuzione Slackware il pacchetto è sul primo cd.
    /mnt/cdrom/slackware/k/kernel-source-2.4.26-noarch-4.tgz

  3. Aggiornare la configurazione del kernel e delle dipendenze:

    # cd /usr/src/linux
    # cp /boot/config-ide-2.4.26.config
    # make oldconfig
    # make dep

    Nota: Non è necessario ricompilare il kernel.

  4. Scaricare gli ultimi drivers per ATI Radeon dal sito web di ATI.

  5. Convertire l' rpm a un ordinario file tar.gz con il comando:

    # rpm2tgz fglrx-"version".rpm
    # cd /
    # tar zxvf fglrx-"version".tgz
    # cd /lib/modules/fglrx/build_mod/
    # chmod +x ./make.sh && ./make.sh
    # cd .. && chmod +x make_install.sh && ./make_install.sh

  6. Eseguire il comando fglrxconfig per effettuare il setup di X:

    # fglrxconfig

  7. Su Slackware 10 come su ogni altra distribuzione che usa xorg invece di XFree86 bisogna creare un collegamento simbolico tra XF86Config-4 e xorg.conf:

    # ln -sf /etc/X11/XF86Config-4 /etc/X11/xorg.conf

    Ora provate ad avviare X, a patto di non aver ricevuto errori nei passi precedenti, dovreste ora avere un' accelerazione 3D funzionante. Verificate questo con:

    % glxinfo | grep render

    You should now see:

    direct rendering: Yes
