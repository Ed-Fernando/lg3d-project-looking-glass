Vi piacerebbe se il vostro Desktop fosse realmente un ambiente 3D? Vi piacerebbe se il vostro database di CD o filmati si trasformasse in un jukebox 3D? Project Looking Glass è basato su tecnologia Java ed ha l'ambizione di fornire all'utente un'esperienza più ricca dell' ambiente desktop e delle applicazioni attraverso finestre 3D e nuove possbilità di visualizzazione. E' un progetto di sviluppo open source basato ed evoluto dal progetto della divisione Advanced Development di Sun Microsystems. Supporta l'esecuzione delle applicazioni bidimensionali esistenti, senza alcuna modifica, in uno spazio 3D, cos- come le API per la gestione delle finestre 3D e lo sviluppo di applicazioni. Al momento l'integrazione delle applicazioni esistenti è supportata per piattaforme Linux e Solaris x86. La piattaforma per lo sviluppo di applicazioni 3D è disponibile per Linux, Solaris e Windows.

Project Looking Glass è nella sua infanzia. Abbiamo bisogno del vostro aiuto per esplorare molte idee nel desktop e nello spazio in cui si muovono le applicazioni. Abbiamo rilasciato il codice sorgente all'intera comunità affinché ne venga esplorato ogni apsetto della tecnologia invece di limitarne l'accesso a pochi privilegiati. Crediamo che questo sviluppo aperto sia un modello eccellente per perseguire questa vasta ed eccitante opportunità . Quindi la vostra partecipazione è ampiamente desiderata, innoviamo l'interfaccia utente del computer!

La tecnologia Java è il linguaggio di sviluppo principale. Noi crediamo che sia una buona scelta specialmente per il guadagno di produttività derivato da un ricco insieme di API di supporto su cui possiamo scrivere. Detto questo, stiamo pianificando un interfacciamento al C++ in modo tale da far interagire direttamente il le applicazioni con il sistema.

Principali Caratteristiche Supportate Per quelli che non possono aspettare...

Attualmente stiamo focalizzando il nostro lavoro nel fornire le funzionalità principali per avviare l'esplorazione del 3D ed il raffinamento della piattaforma iniziale. La piattaforma corrente offre le seguenti caratteristiche principali:

  • Integrazione iniziale alle applicazini X11 (stato dell'integrazione)
  • Piattaforma scenografica client-server
  • Comportamento lato server dell'esecuzione
  • Supporto per un generico evento di sistema
  • Insieme iniziale di API per la creazione di controlli 3D
  • Supporto di utilità per l'animazione delle risposte agli eventi utente

Per iniziare e installare l'ambiente di sviluppo fare riferimento al documento "Installazione iniziale di Project Looking Glass". Se siete utenti Windows, fate riferimento al documento "Eseguire la versione di sviluppo di LG3D su Microsoft Windows".

Una volta ottenuto un ambiente di sviluppo funzionante, dovreste leggere il documento "Guida agli sviluppatori di Project Looking Glass "che fornisce informazioni dettagliate su Project Looking Glass e sulla creazione di applicazioni 3D.
Se siete utenti NetBeans, fate riferimento al documento "Costruire ed eseguire LG3D con NetBeans".
Ed inoltre "I tutorials di Project Looking Glass" su come scrivere applicazioni per LG3D.

Quando sarai pronto per contribuire alla scrittura del codice o della documentazione relativa al progetto, puoi leggere "Contribuire a Project Looking Glass".

Fare Rapporto delle Problematiche

Le informazioni riguardo le problematiche correnti possono essere trovate utilizzando l' Issue tracker (Fare rapporto dei problemi). Puoi cercare problemi e difetti o trovare un problema specifico. Se incontri un problema che non è stato archiviato, ti preghiamo di segnalarlo. Potresti inoltre sfruttare la possibilità di cercare nell'archivio e-mail sul forum di discussione su javadesktop.org per trovare tutte le informazioni relative.

Quando fai rapporto di un problema ti preghiamo di essere specifico riguardo alla configurazione hardware e software del tuo computer e al sintomo che hai riscontrato. Quando riscontri un crash, i seguenti file di log potrebbero essere utili per fornire maggiori informazioni:

  • /var/tmp/lgserver.log
  • /var/log/Xorg.0.log
    Questo log potrebbe essere eccessivamente lungo. Ti preghiamo quindi di prestare attenzione alle linee che sono identificate da (EE), e riportare ogni linea sospetta.
  • hs_err_pid####.log
    Anche questo log potrebbe essere molto lungo. Ti preghiamo di inviare solo le linee comprese tra l'inizio e la linea identificata da
    "--- P R O C E S S ---".

Se sei inesperto nel segnalare un problema, potrebbero esserti utili questi documenti:

  • Una guida passo passo per poter fornire un bug report utile.
  • Introduzione veloce alla segnalazione dei problemi riscontrati

Come Restare Informati

Project Looking Glass si sta evolvendo continuamente. Il miglior modo per rimanere aggiornati è quello di sottoscriversi alla mailing list di Project Looking Glass. Clicca sulla voce Mailing lists nella sezione Project tools della barra laterale nella pagina https://lg3d.dev.java.net/. Ci sono varie mailing lists da scegliere in base al proprio livello di interesse.

  • Se sei interessato nel sapere cosa c'è di nuovo, sottoscrivitii a announce@lg3d.dev.java.net.
  • Se sei uno sviluppatore e vuoi discutere delle problematiche e delle questioni relative a Project Looking Glass, sottoscriviti a interest@lg3d.dev.java.net.
  • Se sei interessato alle questioni riguardanti la licenza, sottoscriviti a lg3d_license@dev.java.net.
