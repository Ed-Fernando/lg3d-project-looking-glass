在Microsoft Windows下运行Project Looking Glass开发版

Project Looking Glass可以在Windows下运行！

Project Looking Glass是采用独立于平台的Java技术编写的，这意味着它可以在 很多种操作系统上运行。本文档即提供了特别的指导，告诉您怎样在 Windows操作系统下运行Project Looking Glass开发版。

首先，需要配置您的运行环境。运行环境正确配置之后，您就可以下载、 安装Project Looking Glass了。

1. 平台要求

在您开始配置Project Looking Glass开发环境之前，请先确定您的系统满足下列 最低要求：

部件 要求
操作系统 Windows 2000 和 Windows XP，Project Looking Glass已经在这两种操作系统上通过测试。
CPU 1.4 GHz，建议更快
RAM 推荐512Mb
显卡 3D 图形加速卡，最低16Mb显存，支持1.2或更高版本的OpenGL。
支持16位或32位色彩深度。

已通过测试的显卡：
  • ATI MOBILITY RADEON
  • NVIDIA GeForce4
如果您在其它显卡上运行Project Looking Glass也获得了成功， 请发邮件告诉我们：interest@lg3d.dev.java.net

如果您在某种显卡上运行Project Looking Glass时遇到了问题，也请发邮件告诉我们：issues@lg3d-core.dev.java.net
硬盘剩余空间 250MB

2. 已知Windows上运行 Project Looking Glass时可能遇到的问题

X11 支持情况

尽管Project Looking Glass支持X11程序，但Windows不支持X11程序，因此 在Windows下的Project Looking Glass上无法运行X11程序。在Project Looking Glass下 运行Windows本地程序的功能目前也尚未实现。然而，您可以开发、运行Project Looking Glass 3D程序，比如那个CD浏览器，以及其它的3D Java程序。

全屏模式

Project Looking Glass桌面无法完全取代Windows桌面，不过您可以通过设置来 让Project Looking Glass窗口占据整个屏幕。详情参见第6部分， "调整Project Looking Glass桌面 的尺寸"。

3. 配置 Project Looking Glass运行环境

必须先安装几个Java组件，然后才能够运行Project Looking Glass。最简单的 办法是按下列清单所列的顺序下载并逐个安装各个组件。

下载组件

  1. Java 2 SDK, 标准版 1.5.0 Beta 2 或更高的版本
    http://java.sun.com/j2se/1.5.0/download.jsp

    注： 请选择 JDK下载，而不是 JRE

  2. Java 3D SDK 1.3.2-build4
    https://j3d-core.dev.java.net/servlets/ProjectDocumentList

    注： 浏览 j3d-core/Experimental_Builds/1.3.2-build4 子目录，下载 java3d-1_3_2-build4-windows-i586.zip

  3. Java Advanced Imaging API (JAI) 1.1.2 JDK
    http://java.sun.com/products/java-media/jai/downloads/download-1_1_2.html

    注： 选中 JDK(TM) Install: Bundle for installation in a JDK，下载。 选中 Windows下的第一 项，jar文件下载。(jai-1_1_2-windows-i586-jar.zip). 不要选 JDK/JRE 下载, 这些文件不能在JDK 1.5.0.下安装。

安装组件

组件下载完毕之后，按下列指导进行安装：

  1. 用拥有“计算机管理员”权限的用户身份登录。

  2. 安装Java 2 SDK:

    - 用Windows资源管理器找到Java 2 JDK安装程序 (jdk-1_5_0-beta2-windows-i586.exe) ，运行这个程序。
    - 按安装程序的提示安装JDK。
    - JDK将安装在C:\Program Files\Java\jdk1.5.0 目录下（DOS 8.3路径为：c:\progra~1\java\jdk1.5.0).
    - 把JDK路径加到PATH参数当中：

    C:\>set PATH="C:\Program Files\Java\jdk1.5.0\bin";%PATH%

  3. 安装Java 3D SDK:

    - 用Windows资源管理器找到Java 3D ZIP文件 (java3d-1_3_2-build4-windows-i586.zip) ，打开这个文件。
    - 将压缩文件的内容释放到临时目录当中。
    - 打开一个DOS命令行窗口，转到下列路径下：C:\Program Files\Java\jdk1.5.0\jre:

    C:\>cd \progra~1\java\jdk1.5.0\jre

    运行jar，释放临时目录里的Java 3D jar文件：

    C:\PROGRA~1\Java\jdk1.5.0>jar xvf c:\<temp>\java3d-1_3_2-build4-windows-i586\j3d-132-build4-win.jar

    其中<temp> 为保存Java 3D ZIP释放出来的文件的临时目录。

  4. 安装Java高级图像API:

    - 用Windows资源管理器找到Java高级图像API压缩文件(jai-1_1_2-windows-i586-jar.zip) ，打开这个文件。
    - 将其中内容释放到一个临时目录当中。
    - 转到该目录下，运行jar，释放jar文件：

    C:\>cd \temp
    C:\temp>jar xvf jai_windows-i586.jar

    - 现在运行JAI安装程序：

    C:\temp>jai-1_1_2-lib-windows-i586-jre.exe

    注意： 安装过程中，不要接受默认的JRE路径 (例如： C:\Program Files\Java\j2re1.4.2_03)，而应该选择下列路径：

    C:\Program Files\Java\jdk1.5.0\jre

    注：如果遇到下列错误信息弹出：

    不能运行16位 Windows程序
    无法找到文件 c:\temp\pft12~tmp\SETUP.EXE (或它的一个组件)。请确定文件名和路径
    正确，而且所需的 链接库都已提供。

    则重启计算机，重新安装程序。

  5. 现在需要设置几个环境变量。

    JAVA_HOME=c:\progra~1\java\jdk1.5.0
    PATH=%JAVA_HOME%\bin;%PATH%

    通过系统属性对话框，可永久设置这些变量：

    - 选择开始->控制面板
    - 选择性能与维护
    - 选择系统
    - 选择[高级] 标签页 [环境变量]
    - 在系统变量下新建一个JAVA_HOME 变量：

      - 点[新建]按钮，输入下列参数：

      变量名：JAVA_HOME
      变量值：c:\progra~1\java\jdk1.5.0

      - 点[确定]

    - 更新路径变量:

      - 选中Path 一行，点 [编辑]
      - 加入下列路径：

        %JAVA_HOME%\bin;

      - 点[确定]

    - 点 [确定]按钮保存环境变量和系统属性 对话框。

4. 安装 Project Looking Glass开发版

到java.net网站上注册

如果您以前未曾在java.net 网站上注册，那么 您需要注册，创建一个帐号：

  1. 浏览http://www.java.net 网页，在上方点注册链接。
  2. 输入您想要的用户名和您的EMail地址，点[注册]按钮。
  3. 您将会收到一封EMail，告诉您如何为自己的帐号设置密码。 设置密码并接受java.net网站条款之后，您就可以登录了。

下载Project Looking Glass安装包

  1. 访问https://lg3d-core.dev.java.net 网页，在侧面导航条项目 工具部分点击项目文档。
  2. 在右侧树形目录上选中lg3d-core/stable_builds目录，会有稳定版本的清单显示出来。
  3. 在形如release_x.y的链接上单击鼠标右键 （其中x.y是编译的版本号，例如：0.5），选择 "另存为.."。选定一个临时目录，把文件保存起来。
  4. 您需要一个解压缩软件，比如WinZip， 才能将gzip压缩的tar文件解压缩，再安装程序。
  5. 先用解压缩软件把下载的文件解压缩，再把tar文档的内容释放出来。WinZip可以 自动完成这两步。
    注：如果您把文件释放到系统盘根目录(C:\)，那么Project Looking Glass 将安装到C:\lg3d路径下。

5. 运行Project Looking Glass

在Windows下，Project Looking Glass只能在一个本地窗口里运行，不能以全屏桌面的 方式运行。

在这种模式下，Project Looking Glass会话是运行在Windows桌面的一个窗口里。 这样您只能运行Project Looking Glass 3D应用程序，比如那个CD浏览器。在 这种Project Looking Glass里无法运行X11程序和Windows程序。

  1. 运行Project Looking Glass时，可打开一个DOS命令行窗口（开始-> 附件->命令提示符），输入下列指令：

    C:\>\lg3d\bin\lg3d-dev

如果您已成功安装了Project Looking Glass，这是您就会看到Project Looking Glass 桌面出现在一个Windows窗口里了。

点击任务条上的CD图标可以打开CD浏览器，这是一个3D演示程序。 您可以把鼠标指到CD架上，点击CD图像，看看这个程序是怎么运行的。 如果您想了解更多操作信息，可在任务条上点击CD浏览器图标左边的图标， 这样就可以打开一个面板，面板上对演示程序所支持的操作进行了介绍。需要 指出，最左边的计算机图标需要X11环境支持，无法在Windows下运行。

6. 调整Project Looking Glass桌面尺寸

Project Looking Glass默认屏幕分辨率为800×600，屏幕尺寸为36cm×28.8cm。 这个设置可能与您的桌面不符。要想满屏运行Project Looking Glass，您需要 设置这些参数，这样Project Looking Glass才能利用整个的屏幕空间。 可按下列步骤进行设置。

  1. 找到文件j3d1x1 ，该文件在下列目录下：

    \lg3d\etc\lg3d\displayconfig

  2. 备份 j3d1x1文件：

    c:\>cd \lg3d\etc\lg3d\displayconfig
    c:\lg3d\etc\lg3d\displayconfig>copy j3d1x1 j3d1x1.orig

  3. 编辑j3d1x1文件，根据 您屏幕的具体情况修改下列参数：

    首先，将屏幕的宽和高换算成米（英尺乘以0.0254即为米的数值，如：11"＝0.2794m）。 默认屏幕大小为14.2"（0.360m）宽，11.3"（0.288m）高：

    (ScreenAttribute center PhysicalScreenWidth  0.360)
    (ScreenAttribute center PhysicalScreenHeight 0.288)

    现在修改WindowSize属性，该属性设定分辨率，默认为800x600:

    (ScreenAttribute center WindowSize           (800 600))

    最后，要想让Project Looking Glass满屏运行不带Windows边框，可把下面这行开头的// 注释标记删掉：

    //(ScreenAttribute center       WindowSize           NoBorderFullScreen)

  4. 现在您再运行Project Looking Glass，就应该是全屏的了。

    C:\>\lg3d\bin\lg3d-dev

终于可以享受Project Looking Glass桌面了，您可以尽情发挥您的想象，构想全新 的3D用户体验！如果已经准备好进行开发工作，您可以跳到Project Looking Glass开发指南， 了解Project Looking Glass开发方面的信息。
