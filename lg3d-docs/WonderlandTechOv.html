Project Wonderland Technical Overview

October 2008

This 8-part video presentation provides a technical overview of Project Wonderland, an open source toolkit for creating 3D virtual worlds. The video begins with a sampling of virtual worlds created by Wonderland open source community members in order to give you a sense of what is possible with the toolkit. Then each of the Sun Labs Wonderland core team members describes a portion of the system, beginning with the current version and then moving on to what's coming up next in the future.
