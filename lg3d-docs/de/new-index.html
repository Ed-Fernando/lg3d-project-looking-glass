Willkommen bei Project Looking Glass!

Download und Installation: Linux/Solaris x86, Windows   


Was, wenn dein Desktop eine echte 3D Umgebung wäre? Was, wenn deie CD oder Filmsammlung zu einer 3D Jukebox würde? Project Looking Glass basiert auf Java Technologie und erforscht die Möglichkeiten eine reichere Benutzererfahrung durch eine dreidimensionale Arbeitsumgebung und Visualisierungsmöglichkeiten auf den Desktop zu gewöhnlichen Anwendungen zu bringen.   Es ist ein Open-Source Entwicklungsprojekt, das aus Sun Microsystems "Advanced Technology" Projekt hervorgig.   Es unterstützt das Ausführen von unveränderten bestehenden Anwendungen in einer 3D Umgebung, stellt aber auch neue APIs für den 3D Window Manager und Anwendungsentwicklung bereit.  Zur Zeit wird die Integration bestehender Anwendungen auf Linux und Solaris x86 Plattformen unterstützt. Die Plattform zur 3D Anwendungsentwicklung ist sowohl für Linux, als auch Solaris und Windows verfügbar.

Projekt Looking Glass ist immer noch in einem sehr frühen Entwicklungsstadium. Wir brauchen deine Hilfe um noch viele Ideen im Desktop- und Anwednugsraum zu erforschen. Wir haben den Project Looking Glass Quellcode der ganzen Entwicklergemeinde zugänglich gemacht, um jeden Aspekt der Technologie erkunden zu können, anstatt den Zugang auf wenige, privilegierte Entwickler zu beschränken. Wir sind der Meinung, dass offene Softwareentwicklung eine exzellentes Modell ist, um diese aufregende und großartige Gelegenheit zu nutzen.  Bring dich also ein und revolutioniere zusammen mit uns die Benutzeroberläche des Computers!

Java Technologie ist die hauptsächliche Entwicklungs-Sprache.  Wir denken, dass dies eine gute Wahl ist, besonders durch die Produktivitätszuwächse, die Java mit sich bringt und das umfassende Set an APIs, auf das wir aufbauen können. Allerdings planen wir auch C++ Bindings, so dass auch andere Anwendungen direkt mit dem System interagieren können.

Unterstützte Schlüsselfeatures

Im Moment konzentrieren wir uns darauf die Schlüsselfunktionalität bereitzustellen, um die Erforschung der dritten Dimension und die Verbesserung der ursprünglichen Plattform anzustoßen. Die aktuelle Plattform unterstützt die folgenden Schlüsselfeatures:

  • Integration existierender Anwendungen (X11 Clients) (Anwendungs-Status)
  • Client-Server Plattform zur Zeichnung der Szenen
  • Serverseitige Verhaltensausführung
  • Support für ein allgemeines Eventhandler-System
  • Grundset aus Kern-APIs um ein 3D-Widget Set zu erstellen
  • Unterstützung für Anwendungen mit animiertem Userfeedback

Für alle, die nicht warten können....

Um einen Einstieg zu finden und eine Entwicklungsumgebung aufzusetzen, lies das "Getting started with Project Looking Glass" Tutorial. Wenn du Windows benutzt, lies "Running the LG3D Developer's Release on Microsoft Windows".

Sobald du einmal eine funktionierende Entwicklungsumgebung hast, solltest du den "Project Looking Glass Developer's Guide" lesen, der detaillierte Infaormationen über Project Looking Glass und das Erstellen von 3D-Anwendungen für dieses Projekt enthält. Wenn due ein Nutzer von NetBeans bist, lies auch "Building and Running LG3D with NetBeans". Die "Project Looking Glass Tutorials" erklären schließlich, wie LG3D-Anwendungen programmiert werden.

Wenn du bereit bist Code oder Dokumentation zu dem Projekt beizutragen, lies "Contributing to Project Looking Glass" um herauszufinden, wie dies möglich ist.

Wie kann man mithelfen?

Denke als ersten Schritt darüber nach ein "Observer" des Projektes zu werden, um das Projekt kennenzulernen, und aktiv am Diskussionsforum teilzunehmen. Bitte bezieht euch auf das Contributing to Project Looking Glass Dokument für Informationen wie man an dem Projekt mitarbeiten kann. Die JavaDesktop Community Governance Guidelines beschreiben generelle Richtlinien für die Teilnahme an JavaDesktop community Projekten, so wie Project Looking Glass.

Fehler melden

Informationen über aktuelle Probleme und Fehler können durch einen Klick auf den Issue tracker in dem Project tools Abschnitt der Sidebar gefunden werden. Man kann allgemein nach Themen und Fehlern suchen oder ein spezielles Problem finden. Wenn dir ein Problem begegnen sollte, dass noch nicht gemeldet wurde, bitte schreite mit gutem Beispiel voran und trage es ein. Es ist nützlich das Email-Archiv des javadesktop.org-Diskussionsforums zu durchsuchen um mit einem Problem verbundene Informationen zu finden.

Wenn du ein Problem meldest, versuche bitte möglichst genaue Angaben über deine Arbeitsumgebung und die Fehlersymptome zu machen. Wenn du einen Absturz melden willst, liefern die folgenden log-Dateien nützliche Informationen:

  • /var/tmp/lgserver.log
  • /var/log/Xorg.0.log
    Dieses Log kann sehr lang sein. Bitte achte auf die Zeile, die mit(EE) beginnt und melde sämtliche verdächtigen Zeilen.
  • hs_err_pid####.log
    Auch dieses Log kann sehr lang sein. Bitte sende nur Zeilen zwischen dem Dateibeginn und der Zeile
    "--- P R O C E S S ---".

Wenn dir das Verfolgen von Fehlern neu ist, nimm dir einen Moment Zeit und überfliege diese einführenden Artikel:

  • A Step by Step Guide to filing a useful bug report
  • A Quick Introduction to Issue Tracking

In Verbindung bleiben

Project Looking Glass wird kontinuierlich weiterentwickelt. Die beste Art und Weise um "up-to-date" mit den neuesten Entwicklungen zu bleiben ist eine der Mailinglisten zu abonnieren. Klicke auf Mailing lists in dem Project tools Abschnitt der Sidebar auf https://lg3d.dev.java.net/. Es gibt verschiedene Mailinglisten, von denen du je nach Interesse wählen kannst.

  • Wenn du dich nur für Neuigkeiten interessierst, melde dich bei announce@lg3d.dev.java.net an.
  • Bist du ein Entwickler und möchtest entwicklungsbezogene Projekt Looking Glass Themen diskutieren, melde dich bei interest@lg3d.dev.java.net an.
  • Wenn du dich für Lizensierungs-Themen interessierst, melde dich bei lg3d_license@dev.java.net an.
