Das Project Looking Glass Developer's Release unter Microsoft Windows laufen lassen

Project Looking Glass läuft unter Windows!

Das Project Looking Glass (LG3D) unter Windows laufen zu lassen, ist sehr einfach :-) Das Kernstück von LG3D wurde mit der Plattform¨bergreifenden Java Technologie entwickelt, was bedeutet das man es unter mehreren Plattformen laufen lassen kann. So läuft es auch unter Windows (allerdings gibt es da einige Enschränkungen). Um die Installation zu vereinfachen, wurde der Installer als eine exe-Datei veröffentlicht. Diese Dokument beschreibt wie LG3D zu installieren ist und wie du es in deinem Windows System startest mit Hilfe des Installers.

1. Systemvoraussetzungen

Bitte stelle als erstes sicher, das dein System die folgenden minimalen Anforderungen erfüllt:

Komponenten Voraussetzungen
Betriebssystem Getestet unter Windows XP und Windows 2000
CPU 1.4 GHz oder schneller empfohlen
RAM 512MB empfohlen
Graphikkarte 3D beschläunigte Grafikkarte, mit mindestens 16MB VRAM und Treibern die OpenGL unterstützen, mit mindestens Version 1.3 oder besser. 16bit oder 32bit Farben.

Getestete Karten:
  • ATI MOBILITY RADEON
  • ATI Radeon 9800 Pro 128MB
  • NVIDIA GeForce4/5200/6600/6800
  • Intel 855GM/915G

Wenn du erfolgreich LG3D mit anderen Karten zum laufen bekommst, oder du mit bestimmten Karten Probleme hast, lass es uns wissen in unserem Diskussionsforum

Festplattenplatz 250MB

Einschränkungen unter Windows

  • Keine Windowstypischen Programme im 3D Bereich ...

    Obgleich LG3D für Linux und Solaris Unterstützung für das Laufen lassen der Systemanwendungen innerhalb des LG3D Desktops einschließt, ist diese Eigenschaft nicht für die Windows Version vorhanden. Das bedeutet, es ist nicht möglich Windowsprogramme innerhalb von LG3D zu starten. Du kannst, wie immer, Programme entwickeln die als LG3D Programm laufen, so wie der CD Betrachter und andere LG3D Java Anwendungen. Mit anderen Worten, die Windows Version wurde entwickelt um Erfahrungen mit LG3D und der Anwendungsentwicklung zu sammeln, aber nicht um den Windows Desktop zu ersetzen.

2. Herunterladen des LG3D Installers

Der LG3D Installer für Windows installiert alles was du benötigstum LG3D laufen zu lassen, einschließlig JRE und Java 3D. Das Paket ist sehr groß, aber es vereinfacht sehr deine Installtion.

Der Installer kann von dem folgenden Link herunter geladen werden:

  • LG3D Release 1.0.0 Beta 1 Windows Installer (19 October 2006)

Lade den Installer mit der exe Datei herunter. Andere Versionen können in der "Stable build section" auf der Binär Pakete Seite gefunden werden.

3. Installing LG3D

Der LG3D Installer für Windows installiert alles was du benötigstum LG3D laufen zu lassen, einschließlig JRE und Java 3D. Es fügt auch einen Ordner "LG3D" in deinem Startmenü hinzu. Beachte das der Installer JRE und Java 3D in einen LG3D spezifischen Ordner kopiert. Damit verändert die Installation deine existierende Javaumgebung nicht und due mußt auch nichts anpassen.

  1. Suche die gespeicherte Installer exe Datei und starte sie.

  2. Windows öffnet ein Dialogfenster mit der Meldung "Security Warning" wenn du die exe Datei startest, wegen "Unknown Publisher". Drücke "Run/ Weiter" in dem Dialogfenster.

  3. Der Installer zeigt dir nun die Lizenzvereinbarungen. Lese sie und klicle dann auf "I Agree" wen du damit einverstanden bist.

  4. Der Installer fragt dich wohin du es installieren möchtest. Wähle den Ordner in den LG3D installiert werden soll, drücke danach auf "Install".

  5. Nun wird der Installationsfortschritt angezeigt. Wenn er fertig ist, drücke auf den "Close/ Schließen" Button.

Nach der erfolgreichen Installation solltest du das "LG3D Project" als Eintrag in deinem Startmenü sehen, in dem der LG3D launcher, uninstaller und eine Verknüpfung zur LG3D website enthalten ist.

4. LG3D starten

Nach der erfolgreichen Installation, kannst du LG3D von deinem Startmenü aus starten:

Start -> Programs -> LG3D Project -> Start LG3D Project

Klicke auf den launcher. Nach einigen Sekungen geht das LG3D Fenster auf. Glückwunsch! Wenn du Probleme hast,schaue bitte in den Problembehebungs s Abschnitt für Hilfe.

Kurzanleitung

Um zu sehen wie du LG3D am schnellsten nutzen kannst, schaue einfach auf das LG3D First Run Document . Bitte beachte das die "X Apps Specific Features" nicht für das Windows Release zur Verfügung stehen.

Entdecke den Looking Glass Desktop und beginne mit den Trämen bon neuen Wegen den Desktop in 3D zu nutzen! Wenn du bereit bist zum Entwickeln, schaue über die Project Looking Glass Developer's Guide Seite für Informationen über das entwickeln mit dem Project Looking Glass.

Einige kurze FAQs

  • Wie kann ich LG3D in der Konsole minimiert starten?
    1. Klicke auf den LG3D launcher mit der rechten Maustaste. Das zeigt dir eine Auswahlliste.
    2. Wähle "Eigenschaften" aus dem Menü. Das bringt dir ein Einstellungsfenster.
    3. Von dem "Shortcut" Tab in dem Fenster, gehe zu "Run:". Wähle "Minimized/ Minimiert" aus der Pull-down Auswahlliste.
    4. Klicke "OK" in dem Einstellungsfenster

  • Wie kann ich LG3D im Volbildmodus starten?
    1. Rechtsklick auf dne LG3D launcher. Das zeigt dir eine Auswahlliste.
    2. Wähle "Eigenschaften" aus dem Menü. Das bringt dir ein Einstellungsfenster.
    3. Von dem "Shortcut" Tab in dem Fenster, gehe zu "Target:" ("Ziel:"). Füge -f an das Ende (nach dem letzten Anführungsstrich). Es sollte dann in etwa so aussehen:
      ...\bin\lg3d-run.bat" -f
    4. Klicke auf "OK"

  • Was kann ich noch in dem LG3D Fenster einstellen?
    • Schau in das Resizing and Configuring LG3D Window Document .

5. Problembehebung

Bildschirmaktuallisierung ist sehr langsam

Die Grafikkarte ist nicht schnell genug um LG3D laufen zu lassen. Wähle "Mindestens (16 bit)" Farben: Rechtsklick auf den Hintergrund ->wähle "Eigenschaften" ->wähle "Einstellungen" in dem "Bildschirmeinstellungen" Fenster ->wähle "Medium (16bit)" in der "Farbtiefe" Auswahl. Du solltest auch prüfen, ob es ein Update fü,r deine Graffikkarten Treiber gibt. Bitte google fü Details.
