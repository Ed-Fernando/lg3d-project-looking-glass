Resizing and Configuring LG3D Window

By default, LG3D brings up a window of 800x600. Also, the relationship between the physical screen size and the size in the LG3D virtual universe is configured in a certain manner (it assumes the screen size to be 36cm by 28.8cm). The following steps show how to configure these settings to fit to your environment.

  1. Locate the file j3d1x1 which should be in the following location under the LG3D installation directory: 
    2. etc/lg3d/displayconfig/j3d1x1
  2. Make a copy of the j3d1x1 file.

  3. Edit the j3d1x1 file and change the following properties to reflect the resolution and size of your display:

    First specify the width and height in meters (multiply inches by 0.0254 to get the size in meters, so 11" = 0.2794m). The default entry specifies a screen that is 14.2" (0.360m) wide by 11.3" (0.288m) high:

    (ScreenAttribute ProjectLookingGlass PhysicalScreenWidth  0.360)
(ScreenAttribute ProjectLookingGlass PhysicalScreenHeight 0.288)

    And change the WindowSize attribute to the size you prefer. By default, the size is set to 800x600:

    (ScreenAttribute ProjectLookingGlass WindowSize           (800 600))
  4. Now, when you start Project Looking Glass, it should reflect the configuration you've made.

  5. If you want to configure the physical screen configuraiton for the fullscreen execution, make the changes in the following file: 
    etc/lg3d/displayconfig/j3d1x1-nbfs

If you encounter any issue after modifications, check out the lgserver.log file which is located under the /var/tmp directory for Linux and Solaris, or in the current directory for Windows.
