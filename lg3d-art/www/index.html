Welcome to Project Looking Glass Art Subproject!

This project is the place for sharing additional model and art assets for Project Looking Glass.

The assets in this project are separated from the main source as they can be rather large. Developers will not require this project in order to build or run lg3d-core, whereas users may want to include them for customizating the environment.
